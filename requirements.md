Project: Smart Campus Navigation System Based on Spring Boot

Tech stack:
- Java 21
- Spring Boot
- Maven
- MySQL
- Thymeleaf
- HTML/CSS/JavaScript
- AMap JavaScript API
- AMap Web Service API
- QWeather API

Core features:
1. Campus POI search
2. Map display and location detail
3. Walking route planning
4. Smart prompts based on time, weather, and opening hours
5. Basic POI management

Constraints:
- Monolith project
- No microservices
- No Vue/React
- No machine learning
- Suitable for undergraduate graduation project
